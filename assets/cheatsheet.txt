  Vim 代码浏览速查表          ( <leader> = 空格键 ;  关闭本窗口: q )
════════════════════════════════════════════════════════════════════════════

  全局搜索 (fzf + ripgrep)
  ────────────────────────────────────────────────────────────────────────
    <space>f      按文件名找文件
    <space>g      全工程全文搜索 (:Rg)
    <space>/      搜索光标下的单词
    <space>l      当前文件内搜索行
    <space>b      切换已打开的 buffer
      · 在 fzf 弹窗里:  输入过滤  ↑/↓ 选择  回车打开
        Ctrl-v 竖直分屏打开   Ctrl-x 水平分屏打开   Esc 关闭

  代码跳转 (LSP)
  ────────────────────────────────────────────────────────────────────────
    gd            跳到定义
    gr            查看所有引用 (谁用到了它)
    gi            跳到实现
    gy            跳到类型定义
    K             悬浮显示文档 / 函数签名
    Ctrl-o        ← 返回上一个位置
    Ctrl-i        → 前进
    [g  /  ]g     上一个 / 下一个 诊断(报错/警告)

  代码结构 & 调用关系 (LSP)
  ────────────────────────────────────────────────────────────────────────
    <space>o      当前文件大纲:类 / 函数 (:CocOutline)
    <space>s      全工程符号搜索
    <space>ci     调用关系:谁调用了它   (incoming calls)
    <space>co     调用关系:它调用了谁   (outgoing calls)

  Git blame (提交人 / 时间)
  ────────────────────────────────────────────────────────────────────────
    行内自动      光标停在某行约 0.3s,行尾灰字显示该行提交人+时间
    <space>gb     整个文件逐行 blame (:Git blame);窗口内回车跳到该 commit
    <space>gl     当前文件的提交历史

  文件树
  ────────────────────────────────────────────────────────────────────────
    <space>e      开 / 关 文件树
    <space>n      在文件树中定位当前文件

  补全
  ────────────────────────────────────────────────────────────────────────
    Tab / S-Tab   在补全菜单中 下 / 上 选择
    回车          确认选中的补全项

  本速查表
  ────────────────────────────────────────────────────────────────────────
    <space>?      随时打开本速查表       q  关闭
